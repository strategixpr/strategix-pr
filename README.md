# Strategix CMS

> Пошаговая инструкция по установке и первому запуску приложения на macOS и Windows.

## Установка на macOS

### 1. Откройте `.dmg` и перенесите приложение в `Applications`
Скчайте и откройте установочный `Strategix CMS_1.0.0_aarch64.dmg`. Перетащите `Strategix CMS` в папку `Applications` (`Приложения`).

![Шаг 1. Перенос приложения в папку Applications](<1 шаг.jpg>)

### 2. Откройте терминал через папку `Applications`
В Finder нажмите правой кнопкой мыши по папке `Applications` и выберите пункт `Новый терминал по адресу папки`.

![Шаг 2. Открытие терминала через Finder](<2 шаг.jpg>)

### 3. Выполните команду в терминале
Введите и выполните следующую команду:

```bash
sudo xattr -r -c "Strategix CMS.app"
```

Если система запросит пароль, введите пароль вашего macOS-пользователя и нажмите `Enter`.

![Шаг 3. Выполнение команды в терминале](<3 шаг.jpg>)

### 4. Запустите приложение
После выполнения команды откройте `Strategix CMS` из папки `Applications`.

## Установка на Windows

### 1. Скачайте установочный файл
Скачайте установщик `Strategix CMS` для Windows в формате `.exe` или `.msi`.

### 2. Запустите установку
Откройте скачанный файл и следуйте шагам мастера установки.

### 3. Подтвердите запуск, если Windows покажет предупреждение
Если появится окно защиты Windows SmartScreen, нажмите `Подробнее`, затем выберите `Выполнить в любом случае`.

### 4. Завершите установку и откройте приложение
После завершения установки запустите `Strategix CMS` через меню `Пуск` или ярлык на рабочем столе.

## Важно

- Команду `sudo xattr -r -c "Strategix CMS.app"` достаточно выполнить один раз после установки.
- Если приложение не открылось с первого раза, закройте его и запустите повторно.
- Убедитесь, что команда выполняется именно для `Strategix CMS.app` в папке `Applications`.
- Если во время установки на Windows система запросит права администратора, подтвердите действие.
- Если ярлык не появился на рабочем столе, приложение можно найти через меню `Пуск`.
